A4 App .Gestión de Tareas

Descripción

A4 App .Gestión de Tareas es una aplicación diseñada para ayudar a los usuarios a gestionar sus tareas de manera eficiente. Permite a los usuarios agregar, completar y eliminar tareas, manteniendo un registro persistente de las mismas mediante el uso de localStorage.

Características

Agregar Tareas: Los usuarios pueden agregar nuevas tareas a través de un campo de entrada y un botón de “Agregar”.
Completar Tareas: Las tareas pueden ser marcadas como completadas, aplicando un estilo de texto tachado.
Eliminar Tareas: Los usuarios pueden eliminar tareas de la lista.
Persistencia de Datos: Las tareas se guardan en localStorage, asegurando que no se pierdan al cerrar el navegador.
Interfaz Intuitiva: Diseño limpio y fácil de usar, con una lista numerada de tareas.

Instalación

Descomprime y Abre el archivo index.html en tu navegador preferido. !Listo!

Uso
Agregar una Tarea:
Escribe la tarea en el campo de entrada.
Haz clic en el botón “Agregar”.
Completar una Tarea:
Haz clic en el botón “Completar” junto a la tarea.
Eliminar una Tarea:
Haz clic en el botón “Eliminar” junto a la tarea.
Estructura del Proyecto
index.html: Contiene la estructura HTML de la aplicación.
app.js: Contiene la lógica JavaScript para la gestión de tareas.
styles.css: Contiene los estilos CSS para la aplicación (si se usa un archivo CSS separado).
